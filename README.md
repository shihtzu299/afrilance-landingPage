# AfriLance Landing Page

Official landing page for **AfriLance**, a Web3 payment infrastructure enabling secure on-chain payments through decentralized escrow and shareable paylinks.

The landing page introduces the AfriLance ecosystem and provides access to the two core applications:

- **Freelance Escrow** — secure milestone-based payments between clients and freelancers.
- **PayLinks** — shareable payment links for receiving stablecoin transfers on-chain.
